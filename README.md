API in this app:
(1)
Method: Get
Url: /api/items/
Action: Get all data

(2)
Method: Get
Url: /api/items/"id"/
Action: Get data using id

(3)
Method: Post
Url: /api/items/
Action: Create new item

(4)
Method: Put
Url: /api/items/"id"/
Action: Update data using by id

(5)
Method: Delete
Url: /api/items/"id"/
Action: Delete item using id
